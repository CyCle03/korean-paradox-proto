# 한국형 Paradox 시뮬 데모 프로토타입

이 프로젝트는 한국형 Paradox 감성의 정치 시뮬레이션을 보여주는 데모다.
세력 균형과 사건 로그를 기반으로, AI 요약이 어떤 연출을 만들 수 있는지 확인한다.
완성된 게임이 아니라 검증용 프로토타입이며, 결과는 seed로 재현 가능하다.
드라마틱 요약과 연대기 요약을 한 화면에서 확인할 수 있다.
UI는 최소 데모 수준으로만 제공된다.

## 핵심 특징

- 세력 기반 정치 시뮬레이션
- 인물 5명 + 이벤트 시스템
- seed 기반 재현성
- 드라마틱 AI 요약 (`/ai/explain`)
- 연대기 요약 (`/ai/chronicle`)

## 데모 실행 방법

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

브라우저: `http://127.0.0.1:8000/`

## 데모 보는 법

1) scenario를 `warlord`, seed를 `42`로 두고 시작
2) Explain 버튼을 눌러 위기 요약(3문장) 확인
3) Chronicle 버튼으로 10년 연대기 요약(6~10줄) 확인

## 예시 출력

위기의 뿌리는 권력 다툼과 치안의 균열이다. 위기의 칼날을 쥔 결정적 행위자는 General이며 심각도 4의 충돌을 부른다. 다음 충돌이 이어지면 파국의 문턱을 넘을 조짐이다.

## 이 프로젝트가 증명하는 것

- LLM을 턴마다 호출하지 않아도 게임 연출에 충분한 요약이 가능하다.
- 로그/메타데이터 기반 AI 요약은 통제 가능한 방식으로 활용할 수 있다.
